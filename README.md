Smart Expense Tracker
  .A desktop application built with Java, JavaFX, and MySQL to manage daily expenses.
  .It provides an easy way to record transactions, track spending habits, and visualize expenses with charts.
  .The project demonstrates database integration, clean UI design, and practical Java application development.
Features
  .Add new expenses
  .View all expenses
  .Filter by date range
  .Category-wise pie chart
  .Export data to CSV

Tech Used
  .Java (JDBC, OOP)
  .JavaFX (UI)
  .MySQL (Database)
